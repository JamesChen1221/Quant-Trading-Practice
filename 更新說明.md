# 更新說明

## 版本 1.1.0 - 專案簡化（2026-02-05）

### 主要更新

#### 1. 移除 RVOL 相關功能

**原因**：
- Alpha Vantage 免費版不支援盤前資料（需付費 $49.99/月）
- Polygon.io 需要付費（$29/月）
- 手動輸入需要每日維護

**決定**：
- 專注於 RSI 和 ADX 計算
- 保持系統完全免費
- 簡化專案結構

#### 2. 刪除的檔案

**API 相關**：
- `alphavantage_premarket_volume.py`
- `polygon_premarket_volume.py`
- `auto_premarket_rvol.py`
- `config_template.py`
- `config.py`

**測試腳本**：
- `test_alpha_vantage.py`
- `diagnose_alpha_vantage.py`
- `test_rvol.py`

**文件**：
- `Alpha_Vantage_設定步驟.md`
- `Alpha_Vantage_重要說明.md`
- `RVOL計算說明.md`
- `盤前RVOL說明.md`
- `盤前成交量輸入指南.md`
- `自動獲取盤前成交量指南.md`

#### 3. 簡化主程式

**變更**：
- 將 `calculate_indicators_simple.py` 重命名為 `calculate_indicators.py`
- 移除所有 RVOL 計算邏輯
- 移除 API 相關導入和配置
- 簡化輸出訊息

**保留功能**：
- ✅ RSI 計算（5天和30天序列）
- ✅ ADX 計算（5天和30天序列）
- ✅ 增量計算
- ✅ Excel 格式保留
- ✅ 批次處理

#### 4. 更新文件

**更新的文件**：
- `README.md` - 移除 RVOL 相關說明
- `使用說明.md` - 簡化為 RSI/ADX 說明
- `執行說明.md` - 更新執行範例
- `requirements.txt` - 移除 requests 依賴
- `.gitignore` - 移除 config.py 規則

### 當前功能

#### 計算指標

- **RSI (相對強弱指標)**
  - 5天序列
  - 30天序列
  - 14天週期

- **ADX (平均趨向指標)**
  - 5天序列
  - 30天序列
  - 14天週期

#### 資料來源

- Yahoo Finance（免費）
- 無需 API Key
- 無請求限制

#### 特色功能

- 增量計算（跳過已有資料）
- Excel 格式保留
- 批次處理
- 完全免費

### 升級指南

如果你已經在使用舊版本：

#### 步驟 1：備份資料

```bash
copy 量化交易.xlsx 量化交易_備份.xlsx
```

#### 步驟 2：更新程式碼

```bash
git pull origin main
```

#### 步驟 3：清理舊檔案

舊的 API 相關檔案已被刪除，無需手動清理。

#### 步驟 4：測試執行

```bash
python calculate_indicators.py
```
